## Facial landmark points detection demo

### Requirements
* OpenCV2
* Python

### Usage
```bash
$ python flploc.py
```

### Keys:
<kbd>w</kbd> - Toggle pupils localization<br/>
<kbd>e</kbd> - Toggle eyes localization<br/>
<kbd>a</kbd> - Toggle facial landmark points detection<br/>
<kbd>q</kbd> - Quit
