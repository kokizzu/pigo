## Talk detection demo

### Requirements
* OpenCV2
* Python

### Usage
```bash
$ python talkdet.py
```

### Keys:
<kbd>w</kbd> - Show/hide detected faces (default Off)<br/>
<kbd>e</kbd> - Show/hide detected pupils (default On)<br/>
<kbd>e</kbd> - Show/hide facial landmark points (default On)<br/>
<kbd>q</kbd> - Quit
